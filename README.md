# heart-disease-prediction
# Kalp Hastalığı Tahmin Sistemi  Bu proje, makine öğrenimi tabanlı bir **Kalp Hastalığı Tahmin Sistemi**dir.   Amaç, açık kaynaklı sağlık verileri üzerinde eğitim yapılmış bir model aracılığıyla, kullanıcıların sağlık verilerini kullanarak kalp hastalığı riskini tahmin etmektir.
